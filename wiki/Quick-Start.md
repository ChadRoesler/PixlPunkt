# Quick Start Guide

Get up and running with PixlPunkt in 5 minutes!

---

## <img src="https://raw.githubusercontent.com/ChadRoesler/PixlPunkt/main/docs/assets/icons/rocket_16.png" width="16"> First Launch

When you first open PixlPunkt, you'll see:
- **Tool Rail** on the left - your drawing tools
- **Canvas Area** in the center - where the magic happens
- **Panels** on the right - layers, palette, properties

---

## <img src="https://raw.githubusercontent.com/ChadRoesler/PixlPunkt/main/docs/assets/icons/document_16.png" width="16"> Create Your First Canvas

1. Press `Ctrl+N` or go to **File → New**
2. Choose a canvas size:
   - **16×16** - Icons, tiny sprites
   - **32×32** - Standard game sprites
   - **64×64** - Detailed characters
   - **Custom** - Any size you want
3. Click **Create**

> **Tip:** Start small! Pixel art shines at low resolutions.

---

## <img src="https://raw.githubusercontent.com/ChadRoesler/PixlPunkt/main/docs/assets/icons/edit_16.png" width="16"> Draw Something

### Select the Brush Tool
- Press `B` or click <img src="https://raw.githubusercontent.com/ChadRoesler/PixlPunkt/main/docs/assets/icons/edit_16.png" width="16"> in the tool rail

### Pick a Color
- Click on a color in the **Palette Panel** on the right
- Or use the **Color Picker** (click the foreground color swatch)

### Start Drawing!
- **Left-click** to draw with foreground color
- **Right-click** to pick colors from the canvas
- **Scroll wheel** while holding `Ctrl` to zoom

### Adjust Brush Size
- Press `[` to decrease size
- Press `]` to increase size

---

## <img src="https://raw.githubusercontent.com/ChadRoesler/PixlPunkt/main/docs/assets/icons/layer_16.png" width="16"> Work with Layers

### Add a New Layer
- Press `Ctrl+Shift+N`
- Or click the <img src="https://raw.githubusercontent.com/ChadRoesler/PixlPunkt/main/docs/assets/icons/add_16.png" width="16"> button in the Layers panel

### Layer Tips
- Draw different elements on separate layers (background, character, effects)
- Use layer opacity for shadows and transparency
- Rename layers by double-clicking their name

---

## <img src="https://raw.githubusercontent.com/ChadRoesler/PixlPunkt/main/docs/assets/icons/save_16.png" width="16"> Save Your Work

### Save as PixlPunkt Project (.pxp)
- Press `Ctrl+S`
- Choose a location and name
- This preserves all layers, animation, and settings

### Export as Image
- Press `Ctrl+E` for **Export**
- Choose format: PNG (recommended), GIF, BMP, etc.
- Set scale if you want to upscale (2x, 4x, etc.)

---

## Essential Shortcuts

| Action | Shortcut |
|--------|----------|
| New Canvas | `Ctrl+N` |
| Save | `Ctrl+S` |
| Undo | `Ctrl+Z` |
| Redo | `Ctrl+Y` |
| Brush Tool | `B` |
| Eraser | `E` |
| Fill | `G` |
| Zoom In/Out | `Ctrl+Scroll` |
| Pan Canvas | `Space+Drag` |
| Pick Color | `Right-Click` |
| Swap Colors | `X` |

---

## <img src="https://raw.githubusercontent.com/ChadRoesler/PixlPunkt/main/docs/assets/icons/play_16.png" width="16"> Quick Animation

1. Open the **Timeline** panel (`T`)
2. Draw your first frame
3. Click <img src="https://raw.githubusercontent.com/ChadRoesler/PixlPunkt/main/docs/assets/icons/add_16.png" width="16"> to add a new frame
4. Draw the next pose
5. Press `Space` to play!
