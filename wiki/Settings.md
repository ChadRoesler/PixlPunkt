## <img src="https://raw.githubusercontent.com/ChadRoesler/PixlPunkt/main/docs/assets/icons/keyboard_16.png" width="16"> Keyboard Shortcuts

### Viewing Shortcuts

1. Go to **Edit ? Preferences ? Keyboard**
2. Browse or search for actions
3. View current assignment

### Customizing Shortcuts

1. Find the action to customize
2. Click the shortcut field
3. Press your desired key combination
4. Click **Apply**

### Resolving Conflicts

If your shortcut is already used:
- Yellow warning appears
- Choose to:
  - **Replace** - Remove from other action
  - **Cancel** - Choose different key

### Import/Export

- **Export** - Save shortcuts to file
- **Import** - Load shortcuts from file
- **Reset** - Restore all defaults

## Settings File Location

Settings are stored in JSON:

| Platform | Location |
|----------|----------|
| Windows | `%LocalAppData%\PixlPunkt\settings.json` |
| macOS | `~/Library/Application Support/PixlPunkt/settings.json` |
| Linux | `~/.local/share/PixlPunkt/settings.json` |

### Reset All Settings

1. Close PixlPunkt
2. Delete `settings.json`
3. Relaunch - defaults are restored

### Backup Settings

Copy `settings.json` to back up your configuration.
