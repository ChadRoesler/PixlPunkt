## Gradient Fill Quick Ref

### Dithering Styles

| Style | Vibe |
|-------|------|
| **Bayer 2×2** | Tiny, subtle |
| **Bayer 4×4** | Classic retro ★ |
| **Bayer 8×8** | Chunky, bold |
| **Floyd-Steinberg** | Smooth, organic |
| **Atkinson** | High contrast, Mac classic |
| **Riemersma** | *Chef's kiss* - best quality |
| **Blue Noise** | Film grain vibes |

### Controls

- **Strength** = How much dither (0% = solid, 100% = full pattern)
- **Scale** = Pattern size (1x = fine, 4x = chunky)

## Layer Effects

| Effect | What It Does |
|--------|--------------|
| Drop Shadow | Shadow behind layer |
