# PixlPunkt Icon Reference

Complete mapping of icons used in PixlPunkt, extracted from source code.

> **Icons exported!** All icons are available as `{name}_16.png` and `{name}_20.png` in this folder.

## Quick Usage

```markdown

<!-- In a table -->
| Icon | Action |
|:----:|--------|

```

## Generating PNGs

Run the export script (requires Inkscape):

```powershell
# Install Inkscape first
winget install Inkscape.Inkscape

# Run the export script
.\scripts\export-doc-icons.ps1

# Export specific icons only
.\scripts\export-doc-icons.ps1 -IconsOnly "play,pause,stop"

# Export at different size
.\scripts\export-doc-icons.ps1 -Sizes 24

# Force re-export
.\scripts\export-doc-icons.ps1 -Force
```

Icons are output to `docs/assets/icons/` as `{name}_{size}.png`.
